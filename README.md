# NDGA Portal

Notre Dame Girls' Academy, Kuje-Abuja runs on a custom multi-role school operations portal built for academic management, finance, CBT, elections, reporting, and student-family access.


NDGA Portal is a custom-built school management system for Notre Dame Girls' Academy, Kuje-Abuja. It handles academics, results, attendance, CBT, finance, elections, student access, and selected learning support.
